# 행운의 룰렛 - 학생 추첨 시스템

학생들을 위한 재미있고 공정한 추첨 시스템입니다. 로또 스타일과 핀볼 스타일의 두 가지 애니메이션을 지원합니다.

## 기능

- **반 관리**: 4, 5, 6학년 반을 생성하고 관리
- **학생 관리**: 각 반에 학생 추가/수정/삭제, 포인트 입력
- **점수 구간 설정**: 3개의 점수 구간 (브론즈/실버/골드) 설정
- **추첨 애니메이션**: 
  - 🎰 **로또 스타일**: 공들이 투명한 원형 추첨기 안에서 회전
  - 🎯 **핀볼 스타일**: 공이 페그 사이로 떨어지며 당첨자 선정

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 사용 방법

1. **반 관리** 페이지에서 학년과 반을 생성합니다
2. **학생 관리** 페이지에서 학생들을 추가하고 포인트를 입력합니다
3. **설정** 페이지에서 점수 구간과 애니메이션 스타일을 설정합니다
4. **추첨** 페이지에서 원하는 점수 구간을 선택하고 추첨을 시작합니다

## 기술 스택

- React 18 + TypeScript
- Vite
- Framer Motion (애니메이션)
- Zustand (상태 관리)
- React Router DOM (라우팅)

## 데이터 저장

모든 데이터는 브라우저의 LocalStorage에 저장됩니다. 같은 브라우저에서 접속하면 데이터가 유지됩니다.
